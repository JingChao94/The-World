# The-World
想要通过文件层级来从不同角度记录生活的一个协作式小说,希望能有很多人参与,在不同板块文件夹下建立属于自己坐标的子文件夹及属于自己的文本来写对应的故事.

## 结构预想
###  按实际地理情况来做初步分级
     亚洲(Asia)
     非洲(Africa)
     欧洲(Europe)
     北美洲(North America)
     南美洲(South America)
     大洋洲(Oceania)
     南极洲(Antarctica)
